## Project Overview

Static HTML portfolio website (andreferraz.com) — a professional landing page positioning André Ferraz as an experienced Software Engineer helping startups. Built with plain HTML and Tailwind CSS v4. No build pipeline or package manager required (Tailwind CLI installed globally via mise).

## Project Structure

All source files live under `src/`. Root directory has only project config files.

```
andreferraz.com/
├── src/
│   ├── index.html          # Single-page entry point
│   ├── input.css            # Tailwind directives + custom theme
│   ├── output.css           # Generated by Tailwind CLI (gitignored)
│   ├── manifest.json        # PWA manifest
│   └── images/
│       ├── andre.png
│       ├── condohub01.png
│       ├── favicon/
│       └── gallery/
│           ├── fulls/
│           └── thumbs/
├── CLAUDE.md
└── .gitignore
```

## Development

- **Dev (watch mode):** `tailwindcss -i src/input.css -o src/output.css --watch`
- **Build (minified):** `tailwindcss -i src/input.css -o src/output.css --minify`
- **Preview:** Open `src/index.html` in a browser, or serve with `python3 -m http.server -d src`

There are no test or lint commands.

## Architecture

- **src/index.html** — Single entry point. Sections: hero, about, skills (6 cards), projects (CondoHub, Shipperly, SkillBridge), contact form, footer.
- **src/input.css** — Tailwind v4 CSS-based config with custom theme (colors, fonts).
- **src/output.css** — Generated by Tailwind CLI, gitignored.

## Key Design Tokens (src/input.css @theme)

- Brand accent: `#49fcd4` (cyan/turquoise)
- Dark backgrounds: `#1a1a2e`, `#0f0f1a`
- Card background: `#16213e`
- Fonts: Source Sans 3 (body, weight 300/400/600), Raleway (headings, weight 700/800)

## External Integrations

- FontAwesome 6 loaded via kit (kit.fontawesome.com)
- Google Fonts loaded via CDN (Source Sans 3 + Raleway)
