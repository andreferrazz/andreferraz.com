# andreferraz.com

Personal portfolio website built with HTML and [Tailwind CSS v4](https://tailwindcss.com/).

## Development

Tailwind CSS CLI is required. Install it globally or via [mise](https://mise.jdx.dev/).

**Watch mode:**

```sh
tailwindcss -i src/input.css -o src/output.css --watch
```

**Build (minified):**

```sh
tailwindcss -i src/input.css -o src/output.css --minify
```

**Preview:**

```sh
python3 -m http.server -d src
```

Then open [http://localhost:8000](http://localhost:8000).

## Structure

```
src/
├── index.html       # Single-page entry point
├── input.css        # Tailwind directives + custom theme
├── output.css       # Generated by Tailwind CLI
└── images/
    ├── andre.png
    └── favicon/
```
